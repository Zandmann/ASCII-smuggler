ASCII Smuggler Playground

Ukrywanie payloadu w Unicode Tag Characters (U+E0000–U+E007F) — niewidoczne w większości fontów, odczytywane przez LLM jako zwykły ASCII. Moduł 4 — AI Security n8n Kurs.

Ten playground jest edukacyjny. Prawdziwe wektory dostarczenia: clipboard injection, markdown rendering, kanały które nie normalizują Unicode (Slack, Discord, niektóre LLM chat UI, SharePoint/Copilot 2024). W M4 pokazujemy technikę ukrycia, nie dostarczenia przez Gmail — dla Gmaila używamy fontsize0_playground.
